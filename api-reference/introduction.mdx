Bfloat is a platform building infrastructure and tooling for developing autonomous web agents, starting first with a reliable and secure headless browser service.

### Getting started

Get started by creating an account on [Bfloat](https://console.bfloat.ai) and creating an API key. Then, connect to the headless browser service using the API key.
